Raggle Windows Documentation

This will not run correctly! There will be bugs! You have been warned!

Steps to start Raggle:

  1. Select Raggle Server from the Raggle Start Menu
  2. Select Raggle Client from the Raggle Start Menu
  3. Wait a few minutes while Raggle updates feeds.

There are all sorts of bugs at the moment. Some feeds don't parse, the UI is counter-intuitive, a bunch of features aren't accessible from the web interface, and there is a complete lack of documentation. But, it's a start.
